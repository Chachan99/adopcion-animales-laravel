# 1. GENERAR APP_KEY:

# 2. CONFIGURAR BASE DE DATOS:
#    - Crea una base de datos PostgreSQL en Render
#    - Ve a la pestaña "Info" de tu base de datos
#    - Copia la "Internal Database URL"
#    - Extrae los componentes:
#      postgresql://usuario:password@host/database
#      DB_HOST = host
#      DB_USERNAME = usuario
#      DB_PASSWORD = password
#      DB_DATABASE = database

# 3. CONFIGURAR APP_URL:
#    - Reemplaza "TU-APP-NAME" con el nombre real de tu aplicación en Render
#    - Ejemplo: https://adopcion-animales-app.onrender.com

# 4. CONFIGURACIÓN EN RENDER:
#    - Ve a tu Web Service en Render
#    - Sección "Environment"
#    - Agrega cada variable una por una
#    - NO incluyas las líneas que empiezan con #

# 5. VERIFICAR DESPLIEGUE:
#    - Después de configurar, haz "Manual Deploy"
#    - Revisa los logs para verificar que no hay errores
